# 1行テキスト

![１行テキスト](./template/text.gif)
その名の通り、１行だけ入力可能な項目です。改行を入れることができないため、比較的短い文を入力する際に適しています。例えば「人名」や「住所」「管理番号」が適しています。

::: tip 利用可能
- 初期値の設定
- 検索の対象
- 入力必須の指定
:::
